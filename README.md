# FE

## docker compose up

### Before run script

#### Step 1: Edit file .env with variables

```
VITE_PORT=

VITE_API_URL=
VITE_API_PATH=

VITE_TIME_INTERVAL=

```

#### Step 2: Run script

```
docker compose up
```

## dev

### Before run script

```
VITE_API_URL=
VITE_API_PATH=

VITE_TIME_INTERVAL=

```

#### Step 2: Install package

yarn
